# ⌚ Master Watch - Premium E-Commerce Experience

Welcome to the **Master Watch** repository! This project is a modern, responsive, and high-performance e-commerce web application built specifically for luxury timepieces and smartwatches. 

## ✨ Key Features

- **Dynamic Product Catalog**: Browse through an extensive collection of Men's, Women's, and Smartwatches.
- **Interactive Cart System**: Add, remove, and update quantities seamlessly with real-time total calculations.
- **Optimized Checkout Flow**: A multi-step checkout process with integrated form validation and a sleek order summary.
- **Responsive Design**: Flawless experience across desktop, tablet, and mobile devices using Tailwind CSS.
- **State Management**: Built-in Context API handles global state for carts and products efficiently.
- **Toast Notifications**: Interactive user feedback via clean, minimal toast notifications.
- **Fast Performance**: Powered by Vite and React for lightning-fast module replacement and optimized builds.

## 🚀 Tech Stack

- **Frontend**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Routing**: React Router v6
- **Icons**: React Icons & Lucide React
- **Notifications**: React Hot Toast & SweetAlert2

## 🛠️ Getting Started

To run this project locally, follow these simple steps:

1. **Clone the repository**
   ```bash
   git clone https://github.com/SiratimMChy/Master-Watch.git
   cd Master-Watch
   ```
2. **Install Dependencies**
   ```bash
   npm install
   ```
3. **Run the Development Server**
   ```bash
   npm run dev
   ```
4. **Build for Production**
   ```bash
   npm run build
   ```

## 📐 Project Architecture

Our project follows a component-driven architecture:
- `src/Pages/`: Contains main route views (Home, Products, Cart, Checkout, etc.)
- `src/component/`: Reusable UI elements (Cards, Navigation, Sliders, Forms)
- `src/context/`: Global state providers (CartContext, ProductContext)
- `src/router/`: Centralized routing configuration
